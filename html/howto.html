Three Ways to Insert CSS

There are three ways of inserting a style sheet:

  • External style sheet
  • Internal style sheet
  • Inline style sheet

Cascading Order

What style will be used when ther is more than one style specified for an HTML element?

All the styles in a page will cascade into a new virtual style sheet by the following reules, where number one has the highiest priority:

  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the head section)
  3. Browser default

So, an inline styloe has the highiest priority, and will override external and internal sytles and browser defaults.
